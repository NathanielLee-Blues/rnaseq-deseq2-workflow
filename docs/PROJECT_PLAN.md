# Project plan

## Project title

RNA-seq Differential Expression Workflow with DESeq2

## Purpose

This project demonstrates a reproducible RNA-seq differential expression workflow using R and DESeq2.

It is designed as a bioinformatics portfolio project showing transcriptomics analysis, statistical testing, visualisation, and interpretation.

## Dataset

The initial dataset will be the Bioconductor `airway` dataset, which contains RNA-seq count data from human airway smooth muscle cells treated with dexamethasone or left untreated.

## Analysis stages

1. Load count data and sample metadata.
2. Export processed count matrix and metadata.
3. Construct a DESeq2 dataset.
4. Run differential expression analysis.
5. Generate QC and exploratory plots.
6. Generate DESeq2 results tables.
7. Visualise results using MA plots, volcano plots, PCA, and heatmaps.
8. Write a short biological interpretation.

## Planned figures

- PCA plot of samples
- MA plot
- volcano plot
- heatmap of top differentially expressed genes

## Portfolio value

This project demonstrates practical experience with gene expression analysis, R, DESeq2, visualisation, reproducibility, and biological interpretation.
